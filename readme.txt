About the project
This project primarily focuses on a store booking system. Users can book a service by selecting various options, and store owners can view these bookings.

Main functions:
1. Booking: Utilizes a linear stepper to guide users through different steps.
2. Service: Displays a price list and map. Clicking on a location opens a navigator, helping users easily find the store.
3. Login: Users must create an account and log in before making a booking.
4. Responsive Page: The site features responsive components to accommodate different screen sizes.
5. Management: Offers CRUD (Create, Read, Update, Delete) functions, allowing users to modify data in MongoDB.

Used API:
1. Maps JavaScript API: https://developers.google.com/maps/documentation/javascript/examples/map-geolocation
2. Geolocation API: https://developers.google.com/maps/documentation/geolocation/overview
